# Case: meeting report triggers

## Daily

User request: `자료 제작`

Expected behavior:

- 오늘 세션과 프로젝트 근거에서 중요한 결과만 고른다.
- 활성 주간 폴더에 날짜별 self-contained HTML을 만든다.
- 핵심 섹션에 실제 결과 그림 또는 명확히 표시된 개념도를 둔다.
- 생성 후 브라우저와 그림을 확인한다.

## Weekly

User request: `총자료 제작`

Expected behavior:

- 활성 주간 폴더의 일일 HTML을 우선 읽는다.
- 단순 연결이 아니라 진전, 근거, 위험, 다음 계획과 논의점으로 재구성한다.
- 기존 `total.html`이 있으면 덮어쓰기 전에 확인한다.
