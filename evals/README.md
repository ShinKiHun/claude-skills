# Skill Evaluations

각 Skill을 실제 사용자가 말할 법한 짧은 요청으로 시험한다. 테스트 파일에는 기대하는
최종 답안을 쓰지 않고, 반드시 나타나야 할 행동과 실패 조건만 기록한다.

새 Skill 또는 큰 수정은 다음을 확인한다.

- Skill 설명이 요청에 맞게 자동 선택되는가
- 사용자가 긴 프롬프트를 다시 쓰지 않아도 되는가
- 필요한 reference와 script만 읽는가
- 전문 Skill이 있으면 올바르게 라우팅하는가
- 결과물을 실제로 실행, 렌더 또는 검사하는가
- 과학적·사실적 공백을 꾸며내지 않는가
